CHURCHES ALIVE

CHURCHES ALIVE is a web application that will allow church congregation during this pandemic period to access church bulletins every week at their favorite Churches with out the fear of contracting or spreading corona virus.

The unique thing about the application is that it does not require congregation to download an app, you just scan a QR code to get access to the web application bulletins.

The new system is set to improve the efficiency of operations of the CHURCH and improve the congregation experience. The application aims to improve congregation experience through;

1. Protecting them from contracting or spreading the corona virus. 2. Offering convenience 3. Providing an interactive system that will help relate with the congregation. 4. Provide better UX/UI experience in the church environment. 5. Offer safe payment options(optional) 6. Fast and interactive message conveyance.

ADVANTAGES OF USING CHURCHES ALIVE.

1. Convenience.

2. Easy Bible and hymns referencing.

3. COST EFFECTIVE. It is cheaper than printing of the weekly Bulletin.

4. No crowds

5. Discreet prayer requests .

HOW THE APPLICATION WORKS



To access the bulletin the congregation will scan the QR code at the entrance of the church and will be automatically connected to the web app, once connected the congregation will access the verses, hymns e.t.c that will be available in the bulletins.
EXCITING FEATURES OF THE APP (UNIQUE FEATURES)
1. The application does not need to be downloaded. 2. Quick response and interactive interface. 3. QR-PAY is an optional feature in the application.
